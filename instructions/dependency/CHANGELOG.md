# 变更日志

## 使用说明

> 记录所有知识库文档的变更历史，AI Agent 可通过此文件快速了解最近的变更上下文。
> 每次文档变更时必须在此追加记录。

## 格式规范

```markdown
## 变更记录

### [{YYYY-MM-DD}] 初始化: 知识库创建
- **变更文件**: 全部文件
- **变更内容**: 创建项目知识库初始版本，包含产品、领域、架构、依赖四大模块
- **变更原因**: 项目启动，建立AI-Native知识库
- **影响范围**: 无（首次创建）
- **操作人**: {初始创建者}

<!-- 后续变更追加在此处，最新的在最上面 -->
```
