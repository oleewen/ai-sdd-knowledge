# 功能域测试场景

按功能域（或限界上下文）组织测试场景，每域一个子目录，目录内包含 `TEST-CASES.md`。

目录结构示例：

```text
TEST-SCENARIOS/
└── {domain-name}/
    └── TEST-CASES.md    # 该域测试用例清单与场景说明
```

*按实际功能域创建子目录并维护测试用例。*
